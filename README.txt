Программа приниимает от пользователя число, после создается массив array с длиной, 
указанной пользователем длины.
Далее применяется функция FillArray(). При её использовании пользователь вручную вводит каждый элемент массива.
После создаётся массив newArray, с идентичной с первым массивом длиной.
После применяется функция NewArray(). Она принимает в качестве аргументов массивы array и newArray.
С помощью цикла for проверяется длина каждого элемента массива array, если она больше или равна 3, 
то это значение добавляется в массив newArray. На выходе получем массив newArray, 
заполенный строками с длинной в более,чем 3 и более символов, и пустыми строками.
Далее выводятся оба массива с помощью функции PrintArray(). Она выводит каждый элемент массива, 
указанного в качестве аргумента, с помощью цикла for, при этом игнорируя пустые строки.
